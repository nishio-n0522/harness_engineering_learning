# Findings

ベンチマーク実行や fixture 検証で得た学びを時系列で記録する。「観察 → 解釈 → 影響」の3層に分けて書くと、後で振り返りやすい。

## 記入テンプレート

```markdown
### YYYY-MM-DD: <タイトル>

- **観察**: 何を見たか（数字や検出有無は具体的に）
- **解釈**: それが何を意味するか
- **影響**: Phoenix V3 / 自分のチームの選定にどう効くか
- **参照**: results/bench-XXXX.md, fixtures/NN-... など
```
